El concepto de matriz

El objeto Array se utiliza para almacenar varios valores en una sola variable:

El primer elemento de la matriz es 0, el segundo es 1, y así sucesivamente.

Method Description  
newArray() Crea un nuevo array con los datos existentes en variables
toString() Convierte el array en una cadena de texto
join() Convierte el array en una cadena de texto y permite cambiar la coma por otro separador
pop() Quita el último elemento del array
push() Añade un nuevo elemento al final del array
shift() Quita el primer elemento del array
unshift() Añade un elemento al inicio del array desplazando a los demás
delete[] Elimina el valor en la posición que elijamos, dejando un hueco (undefinded)
splice() Añade nuevos elemento al array en la posición que nosotros elegimos, perimitiéndonos además, sustituir los valores antiguos por los nuevos o simplemente añadir los nuevos

También podemos seleccionar los elemntos del array indicando su posición [0,1,2.......]
