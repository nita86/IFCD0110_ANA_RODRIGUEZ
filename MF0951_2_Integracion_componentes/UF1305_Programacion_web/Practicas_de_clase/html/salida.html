Salidas de Javascript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando innerHTML.
  • La escritura en la salida HTML usando document.write().
  • Escribir en un cuadro de alerta, usando window.alert().
  • Escribir en la consola del navegador, usando console.log().

Usando innerHTML.

<script>
document.getElementById("demo1").innerHTML=5+6;
</script>

Usando document.write().

<script>
document.write(5+6)
</script>

Al pulsarlo arroja el resultado en otra página, haciendo desaparecer la nuestra. Se usa para pruebas

Usando window.alert().

<script>
window.alert(5+6);
</script>

Usando console.log().

<script>
console.log;
</script>

El resultado se muestra en la consola del navegador (F12, console). Se usa para pruebas

Cuadros de diálogo con ventanas

JavaScript puede mostrar cuadros de diálogo a través de elementkos

  • confirm ()
  • alert ()
  • prompt ()

confirm()

Muestra un cuadro de diálogo con un mensaje especificado, junto con un clic en Aceptar y un botón Cancelar.Pe utiliza si desea que el usuario verifique o aceptar algo.

confirm (mensaje)

alert()

Muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.Se utiliza para asegurarse que la información llega hasta el usuario.

alert (mensaje)

prompt()

Muestra un cuadro de diálogo que solicita al visitante que introduzca un dato. Cuando un cuadro de mensaje aparece, el usuario tendrá que haga clic en "Aceptar" o "Cancelar" para continuar después de introducir un valor de entrada. No se debe abusar de este método, ya que evita que el usuario pueda acceder a otras partes de la página hasta que la caja está cerrada. El método prompt () devuelve el valor de entrada si el usuario hace clic en "OK". Si el usuario hace clic en "cancelar" el método devuelve null.

prompt ("testo del mensaje", "respuesta")

windows open()

Abre una nueva ventana del navegador.

Para cerrarla se usa close()

function abrir() {
   var ventanaNueva = window.open ("", "", "width=200,height=200");
   ventanaNueva.document.write("texto en la nueva ventana");
   }

es muy importante no poner espacios en blanco entre las comillas que definen la variable

windows close()

Cierra la ventana actual.

Consejo: Este método se utiliza a menudo junto con el método open().

var ventana;
function abrirVentana() {
   ventana = window.open ("", "", "width=200,height=200");
   ventana.document.write("texto en la nueva ventana");
   }
function cerrarVentana() {
   ventana.close();
   }

Para cerrar es necesaerio declarar primero la var ventana pero no darle ningún valor.

window.print()

Abre el cuadro de diálogo Imprimir, que permite al usuario seleccionar las opciones de impresión preferidos.

function imprimir() {
   window.print();
   }

setInterval() y clearInterval()

setInterval()llama a una función o evalúa una expresión a intervalos especificados (en milisegundos).Continuará llamando a la función hasta que invoquemos a clearInterval() o la ventana está cerrada.

setInterval(function, milliseconds, param1, param2, ...)

moveBy() y resizeBy()

moveBy() abre una nueva ventana, y la nueva ventana respecto a su posición actual

resizeBy() abre una nueva ventana, y la la cambia de tamaño
