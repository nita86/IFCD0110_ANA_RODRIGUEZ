¿Dónde se inserta JavaScript?

  • En HTML, el código JavaScript en HTML debe ser insertado entre las etiquetas <script> y </script>.
  • Los scripts pueden ser colocados en el <body>, o en la sección <head> de una página HTML, o en ambos.
  • También pueden colocarse en un archivo externo con la extensión .js
  • El código JavaScript puede ser insertado en línea, en las etiquetas HTML dentro del atributo de evento onClick

Ejemplo: JS en <head>

<!DOCTYPE html>
<html>
<head>
    <script>
        function mifuncion() {
        document.getElementById("demostracion").innerHTML ="parrafo cambiado.";
        }
    </script>
</head>
<body>
<h1>A Web Page</h1>
<p id="demostracion">A Paragraph</p>
<button type="button" onclick="mifuncion()">Try it</button>
</body>
</html>

Ejemplo: JS en <body>

<!DOCTYPE html>
<html>
<body>
<h1>A Web Page</h1>
<p id="demostracion">A Paragraph</p>
<button type="button" onclick="mifuncion()">Try it</button>
    <script>
        function mifuncion() {
        document.getElementById("demostracion").innerHTML = "parrafo cambiado.";
        }
    </script>
</body>
</html>

Ejemplo: en archivo externo .js

function mifuncion() {
   document.getElementById("demostracion").style.color = "red";
   document.getElementById("demostracion").innerHTML = "Chanchan";
   }

Ejemplo: JS en dentro de una etiqueta HTML

<p id="demostracion1">JavaScript puede cambiar el contenido HTML</p>
<button type="button" onclick="document.getElementById('demostracion1').innerHTML='Hola JavaScript'">Pulsa aqui</button>
