Lenguaje HTML

HTML, sigla en ingles de Hyper Text Markup Language (lenguaje de marcado de hipertexto), hace referencia al lenguaje de marcado

para la elaboracion de páginas Web. Es un estandar que sirve de referencia del software que conecta con la elaboración de páginas

web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido

de una página web, como texto, imagenes, videos, juegos, entre otros.

HTML, es un estandar a cargo del World Wide Web Consirtium dedicada a la estandarización de casi

todas las tecnologias ligadas a la web, sobre todo en lo referente a su escritura e interpretación
