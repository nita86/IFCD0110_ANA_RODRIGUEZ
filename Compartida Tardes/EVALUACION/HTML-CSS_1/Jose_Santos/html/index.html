CONFECCIÓN Y PUBLICACIÓN DE PÁGINAS WEB

Página Web

Una pagina web ,es un documento capaza de contener

texto,sonido,video,programas,enlaces,imágenes y

muchas otras cosas,adaptada para la llamada World

Wide Web Consorium(W3C) y que puede ser

mostrada mediante un navegador web.Esta

información se encuentra generalmente en

formato HTML o XHHTML , y puede proporcionar acceso

a otras páginas web mediante enlaces de hipertexto.

Sitio Web

Las páginas webpueden estar almacenadas en un

equipo local o en un servidor web remoto.Elservidor

webpuede restringir el acceso únicamente a redes

privadas,por ejemplo,en una internet corporativa,o

puede publicar las páginas en Wold Wide Web.El

acceso a las páginas web es realizado mediante una

transferencia desde servidores,utilizando el protocolo

de transferencia de hipertexto(HTTP).
