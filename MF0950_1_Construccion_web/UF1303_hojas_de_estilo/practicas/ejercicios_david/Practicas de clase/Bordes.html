Bordes CSS

Estilo de borde

El border-style especifica qué tipo de borde a Mostrar.

Se permiten los Siguientes Valores:

  • dotted - Definir punteado Borde.
  • dashed - Definir discontinuo Borde.
  • solid - Definir Sólido Borde.
  • double - Definir Borde doble.
  • groove- Definir Borde ranurado 3D. El efecto Depende del valor del color de del Borde.
  • ridge- Definir Borde acanalado en 3D. El efecto Depende del valor del color de del Borde.
  • inset- Definir Borde de Inserción 3D. El efecto Depende del valor del color de del Borde.
  • outset- Definir Borde de inicio 3D. El efecto Depende del valor del color de del Borde.
  • none - Sin definir Bordes
  • hidden - Definir Borde ocultos.

El border-style puede tener uno de un cuatro Valores (Para El Borde superior, el Borde derecho, el Borde inferior y el Borde Izquierdo).

Ejemplo en CSS:

p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}

Ancho del Borde

El border-width especifica el ancho de los cuatro bordes.

El ancho se puede establecer como un tamaño específico (en px, pt, cm, em, etc.) o utilizando uno de los Tres Valores predefinidos: delgado, medio o grueso.

El border-width puede tener uno de un cuatro valores (Para El Borde superior, el Borde derecho, el Borde inferior y el Borde Izquierdo).

Ejemplo - CSS

p.one {
border-style: solid;
border-width: 5px;
}

p.two {
border-style: solid;
border-width: medium;
}

p.three {
border-style: dotted;
border-width: 2px;
}

p.four {
border-style: dotted;
border-width: thick;
}

p.five {
border-style: double;
border-width: 15px;
}

p.six {
border-style: double;
border-width: thick;
}

p.seven {
border-style: solid;
border-width: 2px 10px 4px 20px;
}

Color del Borde

La border-color utiliza para establecer el color de los cuatro bordes.

El color de Puede Ser establecido por:

  • Nombre: Nombre de color, de Como "rojo"
  • Hex: especifique hexadecimal valor de la ONU, de Como "# FF0000"
  • RGB: especifique RGB valor de la ONU, de Como "rgb (255,0,0)"
  • Transparente

La border-colorpropiedad PUEDE Tener uno de un cuatro Valores (Para El Borde superior, el Borde derecho, el Borde inferior y el Borde Izquierdo). 

Si border-colorNo Está configurado, hereda El color del Elemento.

Ejemplo:

p.one {
border-style: solid;
border-color: red;
}

p.two {
border-style: solid;
border-color: green;
}

p.three {
border-style: solid;
border-color: red green blue yellow;
}

Bordes - Lados individuales

Los ejemplos anteriores se han visto que es posible especificar una frontera diferente para cada lado. En CSS, también hay propiedades para especificar cada uno de los bordes (superior, derecho, inferior e izquierda):

Ejemplo:

p {
border-top-style: dotted;
border-right-style: solid;
border-bottom-style: dotted;
border-left-style: solid;
}

2 different border styles.

Border - Propiedad taquigrafía

Como PUEDE ver en los EJEMPLOS Anteriores, heno Muchas propiedades considerar una Cuando Se Trata de bordes.

Para acortar el Código, también Es Posible especificar TODAS LAS propiedades de borde Individuales En Una propiedad.

La borderpropiedad Es Una propiedad abreviada de las Siguientes propiedades de borde Individuales:

  • border-width
  • border-style (Necesario)
  • border-color

Ejemplo:

p {
border: 5px solid red;
}
