Diseño de cuadrícula

El Diseño de cuadrícula Módulo CSS ofrece un sistema basado en el diseño de la red, con filas y columnas, por lo que es más fácil diseñar páginas web sin tener que utilizar flotadores y posicionamiento.

display:grid

Un diseño de la red consiste en un elemento padre, con uno o más elementos secundarios.

  • grid-template-columns:número y anchura de las columnas (150px auto auto 150px;)
  • grid-template-rows:número y altura de las filas (auto 150px;)
  • grid-template:combina las dos anteriores grid-template-rows / grid-template-columns|grid-template-areas|; (auto 150px / 150px auto auto 150px;)

grid gap

El espacio entre cada columna / fila se denominan gap>.

  • grid-column-gap:separación entre columnas (20px;)
  • grid-row-gap:separación entre filas (10px;)
  • grid-gap:combina las dos anteriores grid-row-gap / grid-colum-gap; (10px/20px;)

  • grid-column-start:en qué línea de columna empieza el item señalado
  • grid-column-end:en qué línea de columna acaba el item señalado
  • grid-column:combina las dos anteriores grid-column-star / grid-column-end
  • grid-row-start:en qué línea de fila empieza el item señalado
  • grid-row-end:en qué línea de fila acaba el item señalado
  • grid-row:combina las dos anteriores grid-row-star / grid-row-end

grid-template-areas

<style>
.grid_padre {
       display:grid;
       grid-template-areas:
       'header header header header header'/*fila 1, con 5 columnas header*/
       'nav main main main aside'/*fila 2 1nav 2main 1aside*/
       'nav footer footer footer footer';/*fila3 1nav 3footer*/
       }

.area1 {grid-area: header;}
.area2 {grid-area: nav;}
.area3 {grid-area: main;}
.area4 {grid-area: aside;}
.area5 {grid-area: footer;}

</style>

<div class="grid_padre">
