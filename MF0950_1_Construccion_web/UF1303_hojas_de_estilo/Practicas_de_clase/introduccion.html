¿Qué es CSS?

  • CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada)
  • CSS describe cómo los elementos HTML se van a mostrar en la pantalla, papel, o en otros medios
  • CSS ahorra mucho trabajo . Se puede controlar el diseño de varias páginas web a la vez
  • Estilo externas se almacenan en archivos CSS

Estructura básica de un documento HTML

Ejemplo

<!DOCTYPE html>
<html lang="es-ES">
<head>
<meta charset="utf-8">
<title>Título de la página></title>
</head>
<body>
<h1> Mi primer encabezado.</h1>
<p> Mi primer párrafo.</p>
</body>
</html>

Explicación de las etiquetas básicas.

  1. La etiqueta <!DOCTYPE html> define el tipo de documento.
  2. La etiqueta <html lang="es-ES> define el elemento raíz de una página HTML.
  3. La etiqueta <head> define los metadatos del documento.
  4. La etiqueta <meta charset="utf-8"> define el juego de caracteres del documento.
  5. La etiqueta <title> especifica un título del documento.
  6. La etiqueta <body> contiene el contenido visible de la página.
  7. La etiqueta <h1> define el encabezado principal. Sólo puede haber una en todo el documento.
  8. La etiqueta <p> define cada párrafo.

Etiquetas HTML.

Las etiquetas HTML son los nombres de elementos entre paréntesis angulares:

<Nombre de etiqueta>contenido ...</nombre de la etiqueta>

Etiquetas HTML
Normalmente vienen en pares como <p> y </p>
La primera etiqueta en un par es la etiqueta de inicio, la segunda etiqueta es la etiqueta de fin
.
La etiqueta de fin se escribe igual que la etiqueta de inicio, pero con una barra inclinada (/) insertada antes de que el nombre de la etiqueta

Nota:La etiqueta de inicio también se llama la etiqueta de apertura y la etiqueta final de la etiqueta de cierre

Navegadores web.

El propósito de un navegador web (Chrome, IE, Firefox, Safari) es leer los documentos HTML y mostrarlos.
El navegador no muestra las etiquetas HTML, pero los utiliza para determinar cómo mostrar el documento:


Versión Año
HTML 1991
HTML 2.0 1995
HTML 3.2 1997
HTML 4.01 1999
XHTML 2000
HTML5 2014

Estructura de una página web

A continuación se muestra una visualización de una estructura de la página HTML:

<html>
<head>
<title>Título de la página</title>
</head>
<body>
<h1>Ésto es un encabezado.</h1>
<p>Ésto es un párrafo.</p>
<p>Ésto es otro párrafo.</p>
</body>

Nota:Sólo el contenido dentro de la sección <body> (el área blanca arriba) se muestra en un navegador.
