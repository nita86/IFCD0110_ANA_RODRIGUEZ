Transformaciones 2D

En este capítulo aprenderá acerca de los siguientes métodos de transformación 2D:

  • El translate() se mueve un elemento desde su posición actual (de acuerdo con los parámetros dados para el eje X y el eje Y).
  • El rotate() hace girar un sentido horario (por defecto) o antihorario según un grado dado.
  • Las scale() aumenta o disminuye el tamaño de un elemento (de acuerdo con los parámetros dados para la anchura y altura).
  • El skewX() deforma un elemento a lo largo del eje X por el ángulo dado.
  • El skewY() deforma un elemento a lo largo del eje Y por el ángulo dado.
  • El skew() deforma un elemento a lo largo de la X y el eje Y por los ángulos dados.
  • El matrix() combina todo el 2D métodos de transformada en una sola.
