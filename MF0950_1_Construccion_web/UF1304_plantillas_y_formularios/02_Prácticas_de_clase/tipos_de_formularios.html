Tipos de entrada HTML

Estos son los diferentes tipos de entrada estandar que se pueden utilizar en HTML:

  • <input type="button">
  • <input type="checkbox">
  • <input type="file">
  • <input type="hidden">
  • <input type="image">
  • <input type="password">
  • <input type="radio">
  • <input type="range">
  • <input type="reset">
  • <input type="submit">
  • <input type="text">

Tipo de entrada de texto

<input type="text">define un campo de entrada de texto de una línea :

input type="text" First name:

Last name:



CODIGO:

<form action="/action_page.php">
First name:<br>
<input type="text" name="firstname">
<br>
Last name:<br>
<input type="text" name="lastname">
<br><br>
<input type="submit">
</form>

<input type="checkbox">define una casilla de verificación .

Casillas de verificación permitir que un usuario seleccione cero o más opciones de un número limitado de opciones.

mac
android
ios
linux

<br>
<input type="checkbox" name="sistema2" value="mac">mac
<br>
<input type="checkbox" name="sistema3" value="android">android
<br>
<input type="checkbox" name="sistema4" value="ios">ios
<br>
<input type="checkbox" name="sistema5" value="linux">linux
<br>
<input type="submit">
</form>

Tipo de entrada Radio

<input type="radio"> Male
Female
Other

<form action="/action_page.php">

<input type="radio" name="gender" value="female"> Female<br>
<input type="radio" name="gender" value="other"> Other<br><br>

Tipo de entrada de color

El <input type="color">se utiliza para campos de entrada que deben contener un color.

En función de soporte de los navegadores, un selector de color puede aparecer en el campo de entrada.

<input type="color"> Select your favorite color:

CODIGO

<form action="/action_page.php">
Select your favorite color:
<input type="color" name="favcolor" value="#ff0000">
<input type="submit">
</form>

Tipo de entrada Fecha

El <input type="date">se utiliza para campos de entrada que deben contener una fecha.

En función de soporte de los navegadores, un selector de fecha puede aparecer en el campo de entrada.

<input type="date">

Birthday:


También puede utilizar el min y max atributos para agregar restricciones de fechas:

<input type="date"> Enter a date before 1980-01-01:


Enter a date after 2000-01-01:

CODIGO

<form action="/action_page.php">
Enter a date before 1980-01-01:<br>
<input type="date" name="bday" max="1979-12-31"><br><br>
Enter a date after 2000-01-01:<br>
<input type="date" name="bday" min="2000-01-02"><br><br>
<input type="submit">
</form>

<input type="datetime-local"> fecha de nacimiento

CODIGO

<form action="/action_page.php">
fecha de nacimiento
<input type="datetime-local" name="bdaytime">

</form>

Tipo de entrada de correo electrónico

El <input type="email">se utiliza para campos de entrada que deben contener una dirección de correo electrónico.

En función de soporte del navegador, la dirección de correo electrónico se puede validar de forma automática cuando se presente.

Algunos teléfonos inteligentes reconocen el tipo de correo electrónico, y añaden ".com" al teclado para que coincida con la entrada de correo electrónico.

<input type="email">

CODIGO

<form action="/action_page.php">
E-mail:
<input type="email" name="email">
<input type="submit">
</form>

Tipo de entrada Número

El <input type="number">define una numérico campo de entrada.

También puede establecer restricciones en lo que se aceptan los números.

El ejemplo siguiente muestra un campo de entrada numérico, donde se puede introducir un valor de 1 a 5:

<input type="number">

CODIGO

<form action="/action_page.php">
Cantidad (entre 1 y 5)
<input type="number" name="quantity" min="1" max="5">
<input type="submit">
</form>

El ejemplo siguiente muestra un campo de entrada numérico, donde se puede introducir un valor de 0 a 100, en saltos de 10. El valor predeterminado que aparece por defecto es 30:

Cantidad

CODIGO

<form action="/action_page.php">
Cantidad:
<input type="number" name="quantity"
min="0" max="100" step="10" value="30">
<input type="submit">
</form>

Tipo de entrada Rango

El <input type="range">define un control para la introducción de un número cuyo valor exacto no es importante (como un control deslizante). Rango por defecto es de 0 a 100. Sin embargo, se puede establecer restricciones en lo que los números son aceptados con el min, maxy stepatributos:

<input type="range">

CODIGO

<form action="/action_page.php" method="get">
Points:
<input type="range" name="points" min="0" max="10">
<input type="submit">
</form>
